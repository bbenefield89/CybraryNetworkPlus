# TCP/IP Model

The TCP/IP Model is a more broad explanation of our OSI Model. While the TCP/IP Model also contains all seven layers of the OSI Model it only shows them in as four separate layers which contains multiple OSI Model layers.

## Mnemonic
- Network Interface
  - Never

- Internet
  - Ingest

- Transport
  - Toxic

- Application
  - Apples

## Layers
- Network Interface
  - Layer 1 - Physical
  - Layer 2 - Data Link

- Internet
  - Layer 3 - Network

- Transport
  - Layer 4 - Transport

- Application
  - Layer 5 - Session
  - Layer 6 - Presentation
  - Layer 7 - Application
